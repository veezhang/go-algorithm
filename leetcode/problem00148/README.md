# [148. 排序链表](https://leetcode-cn.com/problems/sort-list/)

## 题目

给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

进阶：

* 你可以在 `O(nlogn)` 时间复杂度和常数级空间复杂度下，对链表进行排序吗？

示例 1:

![例子1](sort_list_1.jpg)

```c
输入：head = [4,2,1,3]
输出：[1,2,3,4]
```

示例 2:

![例子2](sort_list_2.jpg)

```c
输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]
```

示例 3:

```c
输入：head = []
输出：[]
```
